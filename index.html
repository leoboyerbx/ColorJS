ColorJS Slideshow

Color JS Slideshow

Simple JS/CSS library allowing to create simple slideshows for browsers softwares. No need to work on JavaScript, just create your code in HTML/CSS.
Take the tour.

Simple and smooth transitions

Here is the basic thing you need to kown: just 4 transitions, like the one you've just seen. You can choose the direction the slide takes, and that's all. This alleviates the code and makes sure it will work on any device.

How to use the transitions ?

Just define the class of the slides, for the coming:
ibottom for a slide coming from the bottom.
itop for a slide coming from the top.
iright for a slide coming from the right.
ileft for a slide coming from the left.

And for the leaving: obottom for a slide leaving to the bottom.
otop for a slide leaving to the top.
oright for a slide leaving to the right.
oleft for a slide leaving to the left.

You see ? Just like "in" and like "out"

Individual animations

Like in classic slideshow softwares (PowerPoint, Impress...), you can animate separately each element using CSS3 transform properties.
This feature has been designed to be simple and easy to use.
If you wand an element to have a separated animation, just add the .animate class on it. Then set this position when it's hidden:
#element {
transform: /*transform property*/;
}
Now use the .current class to define properties of the element when the slideshow is on the slide:
#element.current {
tranform: /*changed property*/
}
That's it!
Note: you can animate any CSS property but it's recommended to use transform properties.
